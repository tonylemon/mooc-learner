# mooc-learner

MOOC 课程自动学习工具，自动完成视频课程学习。

## 触发条件

当用户请求以下内容时自动触发：
- "学习MOOC课程"
- "帮我刷课"
- "自动学习视频"
- "完成课程学习"
- "学习网课"

## 工作流程

1. **启动浏览器**
   - 使用 Playwright 启动 Chrome 浏览器
   - 打开目标学习页面

2. **分析学习状态**
   - 检测课程章节完成状态
   - 识别需要学习的章节（非"复习"状态）

3. **自动学习**
   - 逐个进入视频播放页面
   - 监控视频播放进度
   - 自动处理各类弹窗（学习计时、完成提示等）

4. **进度保存**
   - 自动保存学习进度
   - 支持中断后继续

5. **完成通知**
   - 生成学习统计报告
   - 显示完成情况

## 输出格式

- 控制台实时输出学习进度
- 完成学习后显示统计报告

## 依赖

- Node.js 18+
- Playwright
- Chrome/Chromium 浏览器

## 配置选项

可在脚本中修改以下配置：
- `TARGET_URL`: 课程学习页面URL
- `VIDEO.MAX_WAIT_SECONDS`: 最大等待时间
- `BROWSER.HEADLESS`: 是否无头模式运行

## 注意事项

1. 需要保持浏览器窗口可见（首次运行需要登录）
2. 确保网络稳定
3. 学习过程会持续运行直到所有章节完成

## 示例

```
用户: 帮我学习MOOC课程
AI: 正在启动MOOC自动学习工具...
```
